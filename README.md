# tuc-cli [![Build Status](https://travis-ci.org/nodenica/tuc-cli.svg?branch=master)](https://travis-ci.org/nodenica/tuc-cli)

> Get the remaining amount on your TUC card


## Install

```
$ npm install --global tuc-cli
```


## Usage

```
$ tuc --help

  Example
    $ tuc 00759795
    30.00
```
